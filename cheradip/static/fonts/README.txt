Bengali PDF font (optional)
==========================
To render Bengali text correctly in exported PDFs, place a Bengali-capable TTF font here:

  NotoSansBengali-Regular.ttf

Download from: https://fonts.google.com/noto/specimen/Noto+Sans+Bengali

If this file is not present, the app will try Windows system fonts (Vrinda, Nirmala UI).
If none are found, PDFs will use Helvetica and Bengali characters may not display correctly.
